`git checkout @ index.html` Откатить index.html до @  
`git checkout @ -- index.html` то же самое  
`git checkout @~~~~` -- Program.cs //Откатить Program.cs на 4 коммита назад  
`git checkout @~4`  
`git checkout -` Переключиться на предыдующую ветку
